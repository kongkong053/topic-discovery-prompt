## 任务

每天产出 6-10 条候选选题。每条选题必须绑在具体数据上——不能只抛概念。

## 账号定位

「钱花哪儿了」——拆你每天花钱时看到的套路。目标受众：会刷抖音、对消费决策敏感、愿意为认知买单的普通人。

**选题核心标准：两层都要硬**

- **外层（有卧槽感）**：让外行产生「不公平感」或「原来我被骗了」
- **内层（有机制）**：让内行看到「钱流到谁口袋了，怎么流的」

两者缺一不可。外层决定点击，内层决定点赞和评论。没有内层的选题直接枪毙。

判断标准：
1. 外层：能不能让一个从没听过这个话题的人，产生「不公平感」？
2. 内层：能不能说清楚「钱从消费者口袋里出来，经过哪些环节，最后停在谁兜里」？

**机制挖掘三问（每条必答）：**
- 钱从消费者口袋里出来，经过哪些环节？
- 每个环节的人靠什么盈利？（不是卖产品，是靠什么「结构」赚钱）
- 这个结构里，谁是弱势，谁是强势，谁是真正的操盘手？

### 【新】选题结构类型（优先产出后两类）

**类型A · 价格对比型（已饱和，慎用）**
「成本X元卖你Y元」——只说明价格差，不说机制。
这条管线已经生产太多了。如果要写，必须带出「为什么能卖这么贵」的机制。

**类型B · 信息差套利型（推荐）**
利用买卖双方信息不对称赚钱的结构。消费者以为自己做了理性决策，其实是被人设计好的。
核心问题：卖家知道什么消费者不知道？这个信息差是怎么被刻意维持的？

**类型C · 制度套利型（强烈推荐）**
利用规则漏洞或监管滞后赚钱的结构。表面合法，实际是精心设计的法律架构。
核心问题：这个局是怎么从「能成立」变成「大规模复制」的？

**类型D · 心理机制型（强烈推荐）**
利用人的认知偏误设计的收费结构。不靠信息差，靠让你「感觉划算」。
核心问题：商家用了什么心理机制让你掏钱？

## 选题输出格式

每条选题输出以下字段：

```
## 选题N

**类型**：（A/B/C/D，对照上面类型表）

**标题**：（带数字的爆款标题，数字要具体）

**卧槽点**：（用一句话说明看完会让人「卧槽」的具体数字或事实）

**机制**：（必填，钱从消费者口袋里出来，经过哪些环节，最后停在谁兜里）

**选题来源**：哪类来源/哪个具体平台

**核心数据**：（至少2个具体数字支撑这个选题）

**受众入口**：普通人怎么接触到这个话题（买过/见过/踩过）

**案例阶梯**：
- L1钩子：（具体的人+具体的事+具体的钱）
- L2展开：（同类场景罗列，至少2个）
- L3对比：（这个现象vs国外/历史/行业对比）
- L4机制：（钱最后流到谁口袋了）

**适配性**：适合做「钩子→反转→机制→对策」四段式结构的哪个环节

**搜索关键词**：（用于验证选题真实性的搜索词，3-5个）
```

## 质量门槛

- 每条选题必须有至少2个可验证的具体数字
- 标题里必须有至少1个具体价格/比例数字
- 不能写「几百块」「很多人」「差不多的价格」——必须精确
- 「卧槽感」自检：这条选题能让一个完全不懂这个行业的人产生「不公平感」吗？如果不能，重写。
